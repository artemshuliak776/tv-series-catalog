# tv-series-catalog
# Консольна утиліта «Каталог серіалів»
Проєкт виконано в межах навчальної практики. Спеціальність 121 "ІПЗ", 2 курс.

## Функціонал:
- Перегляд усього каталогу у вигляді таблиці
- Додавання нових серіалів із валідацією даних
- Фільтрація за статусом перегляду
- Автоматичний підрахунок середньої оцінки
- Збереження та завантаження даних у форматі JSON

## Запуск:
1. Запустіть `python generate_data.py` для генерації тестової бази.
2. Запустіть `python main.py` для роботи з утилітою.
